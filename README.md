# ShoppingMall
### 쇼핑몰 프로젝트를 위해 사용한 개발환경
프로그래밍 언어: JAVA, JSP, Servlet  
웹 표준: HTML, CSS, JavaScript, JQuery, ajax, Bootstrap  
프레임워크 : SpringFramework 
DBMS: Oracle  
ORM: Mybatis  
Server: Tomcat  
BuildTool: maven  
형상관리툴: Git/Github 

### 쇼핑몰 프로젝트 기획
- 이 프로젝트에서는 액세서리를 판매하는 쇼핑몰을 구상하혔습니다.
- 다른 쇼핑몰(무신사, w컨셉, 발란, farfetch, 등)을 참조하여 진행하였습니다.
